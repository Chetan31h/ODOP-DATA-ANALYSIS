# 📊 ODOP Data Analysis Project

## 📌 Overview
This project analyzes the **One District One Product (ODOP)** dataset to understand product specialization across different states in India.

## 🔍 Objectives
- Analyze state-wise product distribution
- Identify category and sector trends
- Study GI-tag distribution
- Perform data cleaning and preprocessing
- Build a machine learning model for category prediction

## 📊 Key Features
- Data Cleaning (handling inconsistencies & missing values)
- Exploratory Data Analysis (EDA)
- Visualization (heatmaps, bar charts, dashboards)
- Correlation Analysis
- Decision Tree Classifier

## 📈 Key Insights
- Strong correlation between total products and GI-tagged items
- Primary category dominates most states
- Significant variation in product distribution across regions

## 🛠️ Tech Stack
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Scikit-learn
